# Performance Review System

## What is it?
A Java desktop application for managing employee performance evaluations within an organizational hierarchy.

## Features
- Multi-role authentication (Employee, Manager, Admin, Highboard)
- 360° performance reviews across 6 categories with weighted scoring
- Goal setting and tracking
- Skill gap analysis and training recommendations
- PDF report export
- SQLite database persistence

## Tech Stack
- **Language:** Java
- **GUI:** JavaFX
- **Database:** SQLite (sqlite-jdbc)
